Linux-learning

A collection of Linux notes, security lab walkthroughs, and hands-on learning exercises.

About

This repository documents my Linux and cybersecurity learning journey.
I’m currently learning through hands-on labs, challenges, and personal notes, including exercises from OverTheWire.

The goal of this repo is to:

Keep track of concepts, commands, and techniques I learn
Document walkthroughs and solutions in my own words
Build a personal reference for future study and practice
Note on Passwords / Lab Credentials

I will not be posting passwords, flags, or credentials for any labs.

This is intentional because:

I do not want future learners to accidentally see answers or skip the learning process
I want this repository to focus on my notes, methodology, and understanding rather than simply providing solutions
